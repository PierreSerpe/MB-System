Detailed Description

Defines SEG-Y trace header used when extracting seismic reflection or subbottom profiler data from swath data files.

Defines the SEG-Y trace header used by MB-System when extracting seismic reflection or subbottom profiler data from swath mapping data files. The SIOSEIS implementation of the trace header is used because it incorporates a deep water delay value not found in most other implementations.See the SIOSEIS web pages for information:. http://sioseis.ucsd.edu/ http://sioseis.ucsd.edu/segy.header.html
