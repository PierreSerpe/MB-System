34 /*
35  * Notes on the MBF_EM12DARW data format:
36  * 1. This data format is used to store 81 beam SIMRAD EM12 bathymetry
37  * data. This format was created and used by R.B.Owens for
38  * SIMRAD EM12 data collected on the RRS Charles Darwin;
39  * 2. The data consist of 1056-byte records including 1-byte characters,
40  * 2-byte and 4-byte integers, 4-byte floats and 8-byte doubles.
41  * 3. The 81 depth values are stored centered in 81 value arrays. The
42  * center beam is in word 40 of the depth and distance arrays.
43  * 4. The "range" is the two-way pulse travel time for each beam (81 values)
44  * 5. The "reflectivity" is the mean backscattering strength (in dB)
45  * over each beam, corrected for beam patterns, echo sounder
46  * parameters and Lambert's law. In the context of MB-System, the
47  * backscatter values are considered to be amplitudes rather than
48  * sidescan because they are coincident with the bathymetry values.
49  * 6. Comments can be embedded in the data as N-byte ascii strings,
50  * where "func" = 100. Comment lines are stored as 1-byte characters
51  * in the "depth" array. For data, "func" = 150.
52  * 7. Mode; 1 = Shallow, 2 = Deep. Controls scaling factors, e.g. Depth is
53  * scaled 1/0.1 for mode 1 and scaled 1/0.2 for mode 2.
54  *
55  */
