Detailed Description

mb_system functions to initialize and transform between projections and geographic coordinates systems.

Declare mb_system functions used to initialize projections, and then to do forward (mb_proj_forward()) and inverse (mb_proj_inverse()) projections between geographic coordinates (longitude and latitude) and projected coordinates (e.g. eastings and northings in meters). One can also tranlate between coordinate systems using mb_proj_transform(). This code uses libproj. The code in libproj derives without modification from the PROJ.4 distribution. PROJ was originally developed by Gerard Evandim, and is now maintained and distributed by Frank Warmerdam, warme.nosp@m.rdam.nosp@m.@pobo.nosp@m.x.co.nosp@m.m
