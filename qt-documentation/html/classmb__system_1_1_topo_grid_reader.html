Detailed Description

TopoGridReader reads topgraphy/bathymetry data from a file, which can be a GMT grid file or a swath file containing data in any sonar format supported by MB-System.

The data is loaded into vtkPoints vertices and vtkCellArray triangles, which can be accesed by the VTK visualization pipeliine such as the one created by QVtkRenderer.
