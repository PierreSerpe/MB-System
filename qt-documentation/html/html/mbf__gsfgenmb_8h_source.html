24 /*
25  * mbf_gsfgenmb.h defines the data structures used by MBIO functions
26  * to store multibeam data read from the MBF_GSFGENMB format (MBIO id 121).
27  *
28  * Author: D. W. Caress
29  * Date: February 27, 1998
30  *
31  */
32 /*
33  * Notes on the MBF_GSFGENMB data format:
34  * 1. The underlying data format is the Generic Sensor Format (GSF)
35  * developed by Shannon Byrne of SAIC. The GSF format stores swath
36  * bathymetry, single beam bathymetry, and other data.
37  * 2. This MBIO i/o module accesses swath bathymtry data stored in
38  * the GSF format using the gsflib library also written at SAIC.
39  * The gsflib calls translate the data from scaled short integers
40  * (big endian) stored in the file to double values. The sensor
41  * specific values held in the GSF data stream are not
42  * accessed by this module. However, all of the GSF records
43  * and the included information are passed when mb_get_all,
44  * mb_put_all, and the mb_buffer routines are used for
45  * reading and writing.
46  */
