33 /*
34  * Notes on the MBF_MSTIFFSS data format:
35  * 1. The MSTIFF data format is used to store raw sidescan data from
36  * Sea Scan sidescan sonars. This format is a variant of
37  * the TIFF image format with navigation and other information
38  * embedded within the file.
39  * 2. The file structure consists of a bunch of pointers to
40  * data objects at various arbitrary locations within the
41  * file. The header contains a pointer to the location of
42  * the "image file directory", which in turn contains
43  * pointers to the locations of data arrays within the file.
44  * 3. As far as MB-System is concerned, this is a read-only
45  * data format.
46  * 4. The raw sidescan data in the file consists of 1000 pings.
47  * Each ping produces two 512 sample arrays - one for
48  * each side (port and starboard).
49  * 5. The sidescan data is not slant range corrected - the
50  * bottom detect and slant range correction is done on
51  * input by MBIO. The data stored internally by MBIO
52  * is slant range corrected.
53  * 6. The MSTIFF files contain lots of information not used by
54  * MBIO, including images of the data derived from a
55  * realtime display.
56  * 7. Comments are not supported in this format.
57  * 8. Each of the possible data fields is identified by a
58  * unique tag:
59  * Tag Field Type Default
60  * --- ---------- -------
61  * 273 Annotation
62  * 272 AnnotationCount
63  * 279 Annotation2
64  * 278 Annotation2Count
65  * 281 Annotation3
66  * 280 Annotation3Count
67  * 260 BinsPerChannel#
68  * 258 BitsPerBin# 8 bits per sample (2 MSB = 0)
69  * 254 Compression 1 (no compression)
70  * 255 CondensedImage
71  * 301 CreatorVersion
72  * 256 Description
73  * 287 Fathometer
74  * 296 Fathometer2
75  * 286 FathometerCount
76  * 257 History
77  * 263 LeftChannel#
78  * 299 LeftChannel2#
79  * 289 Magnetometer
80  * 288 MagnetometerCount
81  * 291 MagnetometerParms
82  * 303 MagnetometerParms2
83  * 269 Marker
84  * 268 MarkerCount
85  * 277 Marker2
86  * 276 Marker2Count
87  * 284 Marker3
88  * 283 Marker3Count
89  * 295 Marker4
90  * 294 Marker4Count
91  * 307 Marker5
92  * 306 Marker5Count
93  * 267 NavInfo#
94  * 275 NavInfo2#
95  * 282 NavInfo3#
96  * 293 NavInfo4#
97  * 297 NavInfo5#
98  * 308 NavInfo6#
99  * 266 NavInfoCount
100  * 304 NavInterpolationTimeout
101  * 274 PingNavInfo
102  * 264 RightChannel#
103  * 300 RightChannel2#
104  * 261 ScrollDirection
105  * 265 SonarDataInfo#
106  * 292 SonarDataInfo2#
107  * 298 SonarDataInfo3#
108  * 259 SonarLines#
109  * 271 SurveyPlotterImage
110  * 270 SurveyPlotterParms
111  * 290 SurveyPlotterParms2
112  * 302 SurveyPlotterParms3
113  * 305 SurveyPlotterParms4
114  * 262 TimeCorrelation#
115  * 285 Y2KTimeCorrelation#
116  */
