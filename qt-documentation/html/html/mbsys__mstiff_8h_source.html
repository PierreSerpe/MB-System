26  * to store sidescan data read from the MBF_MSTIFFSS format (MBIO id 131).
27  *
28  * Author: D. W. Caress
29  * Date: April 10, 1988
30  *

32 /*
33  * Notes on the MBSYS_MSTIFF data structure:
34  * 1. The MSTIFF data format is used to store raw sidescan data from
35  * Sea Scan sidescan sonars. This format is a variant of
36  * the TIFF image format with navigation and other information
37  * embedded within the file.
38  * 2. The file structure consists of a bunch of pointers to
39  * data objects at various arbitrary locations within the
40  * file. The header contains a pointer to the location of
41  * the "image file directory", which in turn contains
42  * pointers to the locations of data arrays within the file.
43  * 3. As far as MB-System is concerned, this is a read-only
44  * data format.
45  * 4. The raw sidescan data in the file consists of 1000 pings.
46  * Each ping produces two 512 sample arrays - one for
47  * each side (port and starboard).
48  * 5. The sidescan data is not slant range corrected - the
49  * bottom detect and slant range correction is done on
50  * input by MBIO. The data stored internally by MBIO
51  * is slant range corrected.
52  * 6. The MSTIFF files contain lots of information not used by
53  * MBIO, including images of the data derived from a
54  * realtime display.
55  * 7. Comments are not supported in this format.
56  * 8. Each of the possible data fields is identified by a
57  * unique tag:
58  * Tag Field Type Default
59  * --- ---------- -------
60  * 273 Annotation
61  * 272 AnnotationCount
62  * 279 Annotation2
63  * 278 Annotation2Count
64  * 281 Annotation3
65  * 280 Annotation3Count
66  * 260 BinsPerChannel
67  * 258 BitsPerBin 8 bits per sample (2 MSB = 0)
68  * 254 Compression 1 (no compression)
69  * 255 CondensedImage
70  * 301 CreatorVersion
71  * 256 Description
72  * 287 Fathometer
73  * 296 Fathometer2
74  * 286 FathometerCount
75  * 257 History
76  * 263 LeftChannel
77  * 299 LeftChannel2
78  * 289 Magnetometer
79  * 288 MagnetometerCount
80  * 291 MagnetometerParms
81  * 303 MagnetometerParms2
82  * 269 Marker
83  * 268 MarkerCount
84  * 277 Marker2
85  * 276 Marker2Count
86  * 284 Marker3
87  * 283 Marker3Count
88  * 295 Marker4
89  * 294 Marker4Count
90  * 307 Marker5
91  * 306 Marker5Count
92  * 267 NavInfo
93  * 275 NavInfo2
94  * 282 NavInfo3
95  * 293 NavInfo4
96  * 297 NavInfo5
97  * 308 NavInfo6
98  * 266 NavInfoCount
99  * 304 NavInterpolationTimeout
100  * 274 PingNavInfo
101  * 264 RightChannel
102  * 300 RightChannel2
103  * 261 ScrollDirection
104  * 265 SonarDataInfo
105  * 292 SonarDataInfo2
106  * 298 SonarDataInfo3
107  * 259 SonarLines
108  * 271 SurveyPlotterImage
109  * 270 SurveyPlotterParms
110  * 290 SurveyPlotterParms2
111  * 302 SurveyPlotterParms3
112  * 305 SurveyPlotterParms4
113  * 262 TimeCorrelation
114  * 285 Y2KTimeCorrelation
115  */
