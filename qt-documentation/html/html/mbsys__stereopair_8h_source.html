35 /*
36  * Notes on the mbsys_stereopair data structure and associated format:
37  * 1. This is an MB-System i/o module to read and write topography
38  * calculated by photogrammetry from stereo pair photographs.
39  * 2. The structure in mbsys_stereopair.h defines the internal
40  * representation of photogrammetric topography data.
41  * 3. The functions in mbsys_stereopair.c allow for extracting
42  * data from or inserting data into this internal
43  * representation. These functions are called by the
44  * MBIO API functions found in mbio/mb_access.c.
45  * 4. The functions in mbr_photgram.c actually read and
46  * write the mbf_photgram format.
47  * 5. Prototypes for all of the functions in mbsys_stereopair.c
48  * are provided in mbsys_stereopair.h.
49  * 6. This list of functions corresponds to the function pointers
50  * that are included in the structure mb_io_struct that is defined
51  * in the file mbsystem/src/mbio/mb_io.h
52  * Not all of these functions are required - some only make sense to
53  * define if the relevant data type is part of the format. For instance,
54  * do not define mbsys_stereopair_extract_segy() if there are no subbottom
55  * profiler data supported by this data system
56  * 7. The data are structured as deriving from a series of stereo pairs.
57  * The position and attitude of the camera rig are included, as is the
58  * position (relative to the camera) of each sounding derived from the
59  * stereo pair.
60  * 8. Files in format mbf_photgram begin with the characters:
61  * ##PHOTGRAM##V001
62  * Following the 16-byte file header, the individual data records follow
63  * in any order. The defined record types include survey (MB_DATA_DATA),
64  * comment (MB_DATA_COMMENT), and INS (MB_DATA_NAV) which includes
65  * navigation, sensor depth, heading, and attitude sampled more frequently
66  * than the stereo photography.
67  *
68  * Survey data records are binary with the following form:
69  * Number of bytes in record 4U
70  * Data record identifier 4U (0x44445047 = "DDPG" = 1145327687)
71  * Time stamp (MB-System time_d) 8F Decimal seconds since 1970/1/1/ 00:00:00
72  * Longitude 8F Decimal degrees
73  * Latitude 8F Decimal degrees
74  * Sensor depth 8F Meters
75  * Heading 4F Decimal degrees
76  * Roll 4F Decimal degrees
77  * Pitch 4F Decimal degrees
78  * Speed 4F Decimal degrees
79  * Altitude 4F Decimal degrees
80  * N (Number of soundings) 4U
81  * ------------------------------------------------------------
82  * Repeat N times:
83  * ------------------------------------------------------------
84  * acrosstrack 8F meters
85  * alongtrack 8F meters
86  * depth 8F meters
87  * beamflag 1U beamflag
88  * red 1U 0-255
89  * green 1U 0-255
90  * blue 1U 0-255
91  * ------------------------------------------------------------
92  * End identifier 4U (0x454E4421 = "END!" = 1162757153)
93  * Check sum of data record between 2U
94  * and including the data record and
95  * end identifiers
96  *
97  * INS data records are binary with the following form:
98  * Number of bytes in record 4U
99  * Data record identifier 4U (0x4444494E = "DDIN" = 1145325902)
100  * Time stamp (MB-System time_d) 8F Decimal seconds since 1970/1/1/ 00:00:00
101  * Longitude 8F Decimal degrees
102  * Latitude 8F Decimal degrees
103  * Sensor depth 8F Meters
104  * Heading 4F Decimal degrees
105  * Roll 4F Decimal degrees
106  * Pitch 4F Decimal degrees
107  * Speed 4F Decimal degrees
108  * Altitude 4F Decimal degrees
109  * End identifier 4U (0x454E4421 = "END!" = 1162757153)
110  * Check sum of data record between 2U
111  * and including the data record and
112  * end identifiers
113  *
114  * Comment data records are binary with the following form:
115  * Number of bytes in record 4U
116  * Data record identifier 4U (0x4444434D = "DDCM" = 1145324365)
117  * Number of characters in comment 4U Includes at least one terminating
118  * null character, multiple of 4.
119  * Comment NC
120  * End identifier 4U (0x454E4421 = "END!" = 1162757153)
121  * Check sum of data record between 2U
122  * and including the data record and
123  * end identifiers
124  */
