24 /*
25  * mbsys_mr1v2001.h defines the data structures used by MBIO functions
26  * to store interferometry sonard data processed by the Hawaii Mapping
27  * Research Group. This includes data from the MR1, SCAMP, and WHOI
28  * DSL 120
29  *.
30  * The data formats associated with mbsys_mr1v2001 are:
31  * MBF_MR1PRVR2 : MBIO ID 64
32  *
33  * Author: D. W. Caress
34  * Date: March 6, 2003
35  *
36  *
37  */
38 /*
39  * Notes on the MBSYS_MR1V2001 data structure:
40  * 1. The MR1 post processing format uses the xdr external
41  * data representation for portability.
42  * 2. The data stream consists of a file header followed
43  * by individual pings.
44  * 3. The file header contains a comment string and the
45  * number of pings. The comment string is broken up
46  * into multiple comments by MBIO on reading; the comments
47  * are concatenated into a single string on writing.
48  * 4. The pings each contain a header plus the bathymetry and/or
49  * sidescan data.
50  * 6. The data structure defined below includes all of the values
51  * which are passed in the MR1 post processing format.
52  * 7. This code pertains to the post-2001 version of the MR1 format.
53  * 8. This format is used for data processed by HMRG, including
54  * data from the MR1, SCAMP, and WHOI DSL 120.
55  */
